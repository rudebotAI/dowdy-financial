# Dowdy Financial — Equities Trader v4.1

Institutional-grade multi-strategy equity trading engine with 30 equities + 5 sector ETFs.

## Deploy to Railway

### Option A: CLI (fastest)
```bash
cd dowdy-financial
npm install
npm start
```

### Option B: GitHub → Railway
1. Push this folder to a GitHub repo
2. Go to railway.app → New Project → Import repo
3. Deploy (auto-detected as React)

## Local Development
```bash
npm install
npm start
```
Opens at http://localhost:3000
